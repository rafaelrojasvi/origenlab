# Skill: brand-copy

## Purpose

Use this skill when writing or rewriting public-facing OrigenLab copy for pages, sections, CTAs, contact blocks, short descriptions, and small commercial texts.

## When to use

Use this skill for:
- homepage copy
- about page copy
- contact page copy
- category page intros
- service descriptions
- quotation CTAs
- short emails or WhatsApp-ready business wording
- SEO-friendly page summaries

Do not use this skill to invent missing business facts.

## Hard constraints

Never invent:
- brands
- certifications
- official distributor claims
- technical specs
- lead times
- delivery guarantees
- coverage guarantees
- customer logos
- regulatory statements

If information is missing, write copy that stays general and truthful.

## Tone

Write in Spanish.
Tone should be:
- professional
- clear
- technically credible
- commercially useful
- restrained

Avoid hype, fluff, and exaggerated claims.

## Workflow

When using this skill:
1. identify the page/section purpose
2. identify the audience
3. write the clearest truthful version
4. include a practical CTA
5. keep wording concise and scannable

## Good CTA examples

- Solicitar cotización
- Contactar por WhatsApp
- Escribir a contacto@origenlab.cl
- Solicitar información
- Ver categorías de productos

## Output patterns

For a section, provide:
- heading
- subheading or short intro
- body copy
- CTA

For a page, provide:
- title
- meta description
- hero copy
- section copy
- CTA block

## Example guidance

Good:
"Equipos para laboratorio orientados a necesidades de análisis, control y trabajo técnico en distintos entornos profesionales."

Avoid:
"Las soluciones más innovadoras y revolucionarias del mercado mundial."
