# Legacy Mail Migration Notes

Status: historical  
Owner: email-pipeline-maintainers  
Last reviewed: 2026-03-23

Scope: legacy LabDelivery/Outlook preservation and migration context. This is not part of website deployment.

## Preservation-first checklist

1. Export full Outlook mailbox to PST on each legacy machine.
2. Copy original Outlook data files and document source machine/date.
3. Store backups in organized folders and duplicate to a second secure location.
4. Verify PST readability before any cleanup or migration.

## Separation policy

- Keep OrigenLab website operations separate from legacy LabDelivery migration.
- Do not mix credentials, DNS procedures, or runbooks across these domains.
