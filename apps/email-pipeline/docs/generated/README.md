# Generated / internal snapshots

This folder holds **generated or point-in-time operational snapshots** (readiness audits, validation notes, internal plans).

- They are **not** a substitute for current runbooks or architecture docs; see [`../README.md`](../README.md) and [`../../../../docs/DOCUMENTATION_MAP.md`](../../../../docs/DOCUMENTATION_MAP.md).
- Content may include **internal operational detail** (metrics, cohort language, or prioritization). Before publishing this repository as **public** on GitHub, **review, redact, or remove** files here so nothing confidential or customer-specific ships in the open.

Do not commit real mailbox exports, credentials, or raw reports into this folder — those belong outside git (see [`../../.env.example`](../../.env.example) and [`../DATA_LOCATIONS.md`](../DATA_LOCATIONS.md)).
