# Campaign-specific lead automation

Scripts here drive **cohort-scoped** workflows (DR50 reconciliation, ready-8 patches, top-10 sheet application). Versioned DR50 JSON and manifests live in [`data/`](data/).

Normal weekly lead operations do **not** require this folder. Use it when running a documented DR / ready-8 campaign; otherwise prefer the routines in [`../README.md`](../README.md).
